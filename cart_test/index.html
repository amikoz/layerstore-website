Wie funktionieren Payment Links?

Stripe Payment Links ermöglichen es, Zahlungsseiten im Voraus zu erstellen und zu teilen. Diese Links können auf verschiedene Arten genutzt werden:

  • Als direkten Link in E-Mails oder sozialen Medien
  • Als QR-Code für kontaktlose Zahlungen
  • Eingebettet als Kaufbutton auf deiner Website

🔧 Payment Links Implementierung

So erstellst du Stripe Payment Links und integrierst sie in deine Webseite:

  1. Payment Link erstellen: Im Stripe Dashboard unter Payment Links auf "Neu" klicken.
  2. Produkt oder Spende konfigurieren: Entweder ein bestehendes Produkt auswählen, ein neues anlegen oder "Kunden wählt Betrag" auswählen.
  3. Link teilen: Nach der Erstellung kannst du den Link kopieren, einen QR-Code generieren oder einen Kauf-Button für deine Website erstellen.

QR Code generieren: Wähle einen existierenden Link aus und klicke auf "QR Code".

Button für Website erstellen: Wähle einen Link aus und klicke auf "Buy Button". Kopiere den Code und füge ihn in deine Website ein.

<!-- Einfacher Link zu einem Payment Link -->
<a href="https://buy.stripe.com/test_example_link">Jetzt kaufen</a>

<!-- Eingebetteter Buy Button -->
<stripe-buy-button
buy-button-id="buy_btn_YOUR_BUTTON_ID"
publishable-key="pk_test_123"
>
</stripe-buy-button>

<script async src="https://js.stripe.com/v3/buy-button.js"></script>
