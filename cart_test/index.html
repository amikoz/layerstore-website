🔧 Backend Implementierung

Dein Server muss eine Checkout Session erstellen. Hier ist der benötigte PHP-Code:

<?php // create-checkout-session.php require_once 'vendor/autoload.php'; // Stripe Secret Key $stripe = new \Stripe\StripeClient('sk_test_...'); // Empfangene Daten $input = json_decode(file_get_contents('php://input'), true); // Checkout Session erstellen $session = $stripe->checkout->sessions->create([ 'line_items' => $input['line_items'], 'mode' => 'payment', 'success_url' => $input['success_url'], 'cancel_url' => $input['cancel_url'], 'customer_email' => $input['customer_email'] ?? null, 'locale' => 'de', ]); echo json_encode(['url' => $session->url]);

Installation:
composer require stripe/stripe-php
